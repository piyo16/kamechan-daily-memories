# セットアップ手順

## 1. まず使ってみる(1人・1台で使う場合)

なにも設定しなくても使えます。

- PCの場合: このフォルダで `python3 -m http.server 8000` を実行し、ブラウザで
  http://localhost:8000 を開く
- スマホで使う場合: 後述の「GitHub Pagesで公開」をすると、スマホのブラウザで
  開けるようになります。ブラウザのメニューから **「ホーム画面に追加」** すると、
  アプリのように使えます(オフラインでも起動します)

データは端末の中(ブラウザのlocalStorage)に保存されます。

> ⚠️ 端末内保存だけの場合、ブラウザのデータ消去で記録も消えます。
> 「設定」タブの **書き出し(JSON)** で時々バックアップするか、
> 下のスプレッドシート同期を設定してください。

## 2. GitHub Pagesで公開する(スマホから使う・無料)

1. このリポジトリをGitHubにpushする
2. GitHubのリポジトリページで **Settings → Pages** を開く
3. Source を「Deploy from a branch」、Branch を `main` / `(root)` にして Save
4. 数分後に `https://<ユーザー名>.github.io/kamechan-daily-memories/` で開けます

> リポジトリがPublicの場合、アプリのページ自体は誰でも開けますが、
> **記録データは各端末とあなたのスプレッドシートにしかない**ので、
> URLを知られても過去の記録は見えません(合言葉も必要です)。

## 3. 家族と共有する(Googleスプレッドシート同期・無料)

家族全員のスマホで同じ記録を見るための設定です。10分くらいで終わります。

1. [Google スプレッドシート](https://sheets.google.com) で新しいスプレッドシートを作る
   (名前は「かめちゃん日記」など何でもOK)
2. メニューの **拡張機能 → Apps Script** を開く
3. 左側の **⚙️(プロジェクトの設定)** →
   「**「appsscript.json」マニフェスト ファイルをエディタで表示する**」にチェックを入れる
4. エディタに戻り、ファイル一覧の `appsscript.json` を開いて、中身を
   [`gas/appsscript.json`](../gas/appsscript.json) の内容で全文置き換える
   (これで権限が「このスプレッドシートのみ」「このスクリプトが作ったファイルのみ」に絞られます)
5. `コード.gs` を開き、最初から入っているコードを全部消して、このリポジトリの
   [`gas/Code.gs`](../gas/Code.gs) の中身を貼り付けて保存する
6. 左側の歯車(プロジェクトの設定)→ **スクリプト プロパティ** で
   プロパティ `TOKEN`、値に **家族だけが知っている合言葉**(例: `kamechan-himitsu-1234`)を追加
7. 右上の **デプロイ → 新しいデプロイ** を選ぶ
   - 種類: **ウェブアプリ**
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**
   - 「デプロイ」を押して、Googleアカウントの承認を許可する
8. 表示された **ウェブアプリのURL**(`https://script.google.com/macros/s/…/exec`)をコピー
9. かめちゃん日記アプリの **設定タブ** に、URLと合言葉を入力して「いますぐ同期」

家族それぞれのスマホで同じURLと合言葉を設定すれば、全員の記録が
スプレッドシートに集まり、お互いの記録も見えるようになります。

> 💡 デプロイ時の承認画面で「このアプリは確認されていません」と出たら、
> 「詳細」→「(プロジェクト名)に移動」を選べば進めます。自分で作った
> スクリプトなので問題ありません。

### 仕組みと安全性

- 記録はスプレッドシートの `records` シートに1行ずつたまります
  (スプレッドシート自体は**あなたのGoogleアカウントの中**にあり、非公開のままです)
- 「今日のかめ」の写真は、あなたのGoogleドライブのフォルダ
  「かめちゃん日記_photos」に保存されます(こちらも非公開のまま。
  家族の端末へはこのAPI経由で合言葉つきで配られます)
- APIのURLは推測できない長さですが、念のため合言葉(TOKEN)が一致しないと
  読み書きできないようにしています
- 電波がないときの記録は端末に貯まり、次にネットにつながったとき自動で送られます
- 万一の保険として、スプレッドシートは Google の
  「ファイル → 変更履歴」からいつでも過去の状態に戻せます

### Code.gs を更新したとき

コードを貼り直したら、**デプロイ → デプロイを管理 → 編集(鉛筆)→
バージョン: 新バージョン → デプロイ** としてください(URLは変わりません)。
