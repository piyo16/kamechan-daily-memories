# kamechan-daily-memories 🐱

猫のかめちゃんの毎日のごはん・水・おやつを記録するWebアプリ(PWA)。
紙の記録ノートの置きかえとして、スマホから数タップで記録でき、
家族で共有でき、データを安全に長期保存できます。

## できること

- **きろく**: ごはん🍚 / 水💧 / おやつ🍬 を「あげた量・残した量」で入力(摂取量は自動計算)。
  お皿が複数あるときは行を追加でき、合算値も自動計算
- **フードの登録**: よく使うフードを名前+いつもの量+カロリー(100gあたり)で登録。
  名前をドロップダウンで選べて量が自動入力される。登録内容は家族と同期される
- **カロリー**: 登録フードのカロリーから、お皿ごと・今日の合計のkcalを自動計算
- **今日のかめ**: 写真+ひとことメモ+体重+体温を日記として残せる
- **トイレ記録**: おしっこ・うんちをワンタップで記録
- **グラフ**: 年月を選んで(デフォルトは今月)、食事・水・おやつ・カロリー(合計/
  ごはん/おやつ)・体重・体温を週ごと/日ごとの線グラフで表示+週ごとの平均表。
  グラフごとに表示/非表示を切り替えられる
- **りれき**: カレンダーで日付を選んで、その日の記録を確認・削除。
  複数選択・全選択でまとめて削除できる
- **きせかえ**: 背景の色を6種類から選べる(端末ごとの設定)
- **マイページ**: 猫のプロフィール(名前・性別・年齢・猫種・病気)と簡易けんこうチェック
- **AIに相談**: 直近の記録とプロフィールをまとめた相談文を自動生成
  (コピーして無料のAIチャットに貼り付ける方式。API課金なし)
- **家族と共有**: Googleスプレッドシート+Googleドライブ同期(無料・[設定手順](docs/setup.md))
- **バックアップ**: JSON書き出し・読み込み
- **PWA**: ホーム画面に追加すればアプリのように使え、オフラインでも記録できる

## 使い方

セットアップ手順(ローカル起動 / GitHub Pages公開 / 家族共有)は
**[docs/setup.md](docs/setup.md)** を見てください。

## 構成

| パス | 役割 |
|---|---|
| `index.html` `css/style.css` | 画面(ビルド不要・依存ライブラリなし) |
| `js/core.js` | 集計・マージなどの純粋ロジック(Nodeでテスト可能) |
| `js/storage.js` | localStorage保存・スプレッドシート同期・バックアップ |
| `js/chart.js` | SVGトレンドグラフ(クロスヘア+ツールチップ付き) |
| `js/app.js` | 画面制御 |
| `gas/Code.gs` | Googleスプレッドシート側の同期API(Apps Script) |
| `sw.js` `manifest.webmanifest` `icons/` | PWA(オフライン対応・ホーム画面アイコン) |
| `test/core.test.js` | ロジックのテスト(`node test/core.test.js`) |

## データの持ち方

- 記録は常に**端末内(localStorage)が第一**。オフラインでも動く
- 同期を設定すると、記録はGoogleスプレッドシートに1行ずつ蓄積され、
  家族の各端末とマージされる(同一IDは更新日時の新しい方が勝つ)
- 削除は論理削除(トゥームストーン)なので、端末間で削除も正しく伝わる
